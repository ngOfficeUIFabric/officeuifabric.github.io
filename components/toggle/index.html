Components Toggle

The toggle control is a checkbox control that is rendered like a toggle switch. THe Office UI Fabric provides for the ability to have the text describing the toggle control to be placed above or beside the control.

Back to top

Usage

Include a reference to the fabric.css and fabric.components.css files, or the corresponding minified version fabric.min.css and fabric.components.min.css. These are available from the public CDN:
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.min.css" />
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.components.min.css" />

Text Above

The toggle control is an <input/> control with a <label/> and a <span/> that describes the control. This is wrapped within a <div/> that is decorated with a .ms-Toggle class. Add the class ms-Toggle--textAbove to have the text appear above the toggle control.

See the Pen Office UI Fabric - Toggle by Andrew Connell (@andrewconnell) on CodePen.

Back to top

Text Beside

Conversely use the ms-Toggle--textBeside class to place the text beside the control.

See the Pen Office UI Fabric - Toggle (Text Beside) by Andrew Connell (@andrewconnell) on CodePen.
