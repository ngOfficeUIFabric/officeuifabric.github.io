Components Contextual Menu

The Office UI Fabric contains a contextual menu, or a right-click menu, that can be used on pages or within other controls like the Nav Bar control.

Back to top

Usage

The contextual menu is implemented as a <ul> list decorated with the .ms-ContextualMenu class. It isn't visible by default but when the css class .is-open class is added to the <ul> element, the menu is made visible.

Each list item is implemented as a <li> decorated with the CSS class .ms-ContextualMenu-item class. You can indicate if an item is currently selected by setting the .is-selected class on the nested <a> tag, or mark it as disabled using the .is-disabled class.

Headings and Multi-select

Add a non-selectable heading to the menu for groups of items, typically seperated with dividers, by adding another <li> item and add the CSS class .ms-ContextualMenu-item--header. In addition you can make the menu items multi-selectable by adding the CSS class .ms-ContextualMenu--multiselect which lets you pick check-marks next to multiple list items. This class should be added to the top-level <ul> element.
