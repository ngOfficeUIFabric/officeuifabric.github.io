Components List Item

Everywhere you look in Office, Office 365 and SharePoint you will find lists. A list is a collection of list items. This page will cover the details and options for the list item control where the list is covered on another page.

Back to top

Usage

Include a reference to the fabric.css and fabric.components.css files, or the corresponding minified version fabric.min.css and fabric.components.min.css. These are available from the public CDN:
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.min.css" />
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.components.min.css" />

Standard List Item

The core list item is comprised of a handful of <div> elements wrapped within a <div> decorated with the CSS class .ms-ListItem. Within the wrapper <div> developers can add three levels of text, a group of action icons that are aligned in the upper-right corner and some meta text. List Item with Image

You can also take the standard list item and add an image to it, left aligned. This is done by adding another <div> before any text with the CSS class .ms-ListItem-image and then adding an image that's 70px by 70px.

Unread List Item

You know how in Outlook there's a little solid-colored vertical bar on the left-hand side of the list item when it is unread? The Office UI Fabric supports this as well. This is done by adding a .is-unread CSS class to the container <div>.

See the Pen Office UI Fabric - List Item (Unread) by Andrew Connell (@andrewconnell) on CodePen.

Unseen/Flagged List Item

Another option is to flag or mark an item as unseen. This adds a little dog-ear like colored element to the top-left corner of the list item. To do this just add the .is-unseen CSS class to the container <div>.

Selectable List Item

The Office UI Fabric also supports making the list items selectable. To do this you need to add the CSS class .is-selectable to the container <div>. In addition you will need to add another <div> into the container that will be hidden (using the provided .ms-ListItem-selectionTarget class and another class that is wired up to a JavaScript handler to provide the interactivity when the item is clicked.

To select the item, hover over the area where the check would appear and click the empty box that appears. You can also mark an item as selected by adding the CSS class .is-selected to the container <div>.

You will want to add JavaScript to toggle the selected check indicator. An example of this code, using jQuery, can be found in the Office UI Fabric component distribution: Jquery.ListItem.js. This also means that you must include a reference to jQuery as the JQuery.ListItem.js depends on the presence of jQuery.

See the Pen Office UI Fabric - List Item (Selectable) by Andrew Connell (@andrewconnell) on CodePen.
