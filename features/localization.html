The Office UI Fabric includes lots of support for localization such as right-to-left support and language-optmized fonts.

Right-to-Left Support

The Office UI Fabric inclides an alternate CSS file to be included in pages that are written in right-to-left (RTL) languages. The CSS file changes the direction of the columns the responsive grid.

To apply the RTL version of the Office UI Fabric, add the dir attribute to the HTML tag and the RTL version of the CSS file Office UI Fabric uses.

Language-Optimized Fonts

The Western European character set of Segoe UI is used by default by the Office UI Fabric. Languages with other characters will get a version of Segoe UI with a different character set or use a system font.

The HTML lang attribute specifies the language of the element's content. This is typically applied to the root HTML element, where it will be inherited by the entire page as the following example:

<html lang="th-TH">...</html>
