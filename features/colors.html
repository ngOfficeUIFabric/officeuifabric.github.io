Features Colors

The Office UI Fabric offers developers up a collection of colors that they can use to have their application have a familiar look to the Office Design Language, Office 365, Office clients and SharePoint. This is done with four different types of colors:

  • Theme Colors: These are the primary colors used in Office 365, Office and SharePoint. Each product may adopt different colors, however when your application uses the theme colors, if it includes the suite-wide themeing system is enabled in your application these theme classes will pickup the colors for the user's chosen theme. In other words, while one user's theme may be blue, another user may have selected red. If your application uses the theme classes, it would be blue for one user and red for another.
  • Neutral Colors: These are colors that come in 10 (not 50 ) different shades of gray.
  • Accent Colors: These colors are just that, accents. Use these to bring your app to life.
  • Action Colors: These are colors that represent actions like success, error, alerts or information.

Usage

Using the different colors is as simple as adding a CSS class to an element. Each color is provided as a font color, background color & border color. For instance here's a color swatch for the primary theme which is the hex code #0078D7:

<div class="ms-font-su">Office UI Fabric Primary Theme Color Swatch</div>
<div style="border:1px gray solid; padding:10px; width:75%">
  <div class="ms-fontColor-themePrimary">ms-fontColor-themePrimary</div>
  <div class="ms-bgColor-themePrimary ms-fontColor-white" style="width:90%; margin-top:5px; padding:10px;">ms-bgColor-themePrimary</div>
  <div class="ms-borderColor-themePrimary ms-fontColor-themePrimary" style="border-width:1px; border-style:solid; width:90%; margin-top:5px; padding:10px;">ms-borderColor-themePrimary</div>
</div>
