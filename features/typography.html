Features Typography

The Office UI Fabric includes the following ten base font CSS classes and multiple four helper that reflect the typography for the Office Design Language.

Note all fonts in the Office UI Fabric are subject to the terms of the following license: Segoe UI and Fabric CDN License.

Usage

Include a reference to the fabric.css file, or the corresponding minified version fabric.min.css. These are available from the public CDN:
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.min.css" />

CSS Class Font Size Font Family When to Use?
.ms-font-su 42px Segoe UI Light Immersive hero moments, first-run hero moments, and theming.
.ms-font-xxl 28px Segoe UI Light Reading pane subjects, calendar date headers, wayfinding/page titles, people alphabet index, dialog/panel titles.
.ms-font-xl 21px Segoe UI Light Dalog titles, sub-headings, large persona names, hero buttons, callout/peek headings.
.ms-font-l 17px Segoe UI Semilight Global nav, large numerics (month view) large pivots, primary actions (not uppercase).
.ms-font-m-plus 15px Segoe UI Regular Mobile body text, text for most core controls, small pivots, numeric pagination.
.ms-font-m 14px Segoe UI Regular Body text, text for most core controls, small pivots, numeric pagination.
.ms-font-s-plus 13px Segoe UI Regular Mobile examples, form headers, job titles/status in persona controls and grid and table data.
.ms-font-s 12px Segoe UI Regular Mobile examples and preview text or seondary lines in listviews.
.ms-font-xs 11px Segoe UI Regular Mobile examples and uppercase section headers.
.ms-font-mi 10px Segoe UI Semibold Use sparingly in tight spaces where text is mandatory but readability is optional.
