Components Search Box

The search box control is used to let users search an application. It leverages jQuery to implement the interactive functionality.

Back to top

Usage

Include a reference to the fabric.css and fabric.components.css files, or the corresponding minified version fabric.min.css and fabric.components.min.css. These are available from the public CDN:
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.min.css" />
<link rel="stylesheet" href="//appsforoffice.microsoft.com/fabric/1.0/fabric.components.min.css" />

You will want to add JavaScript to clear out the placeholder when the control receives focus as shown in the following snippets. An example of this code, using jQuery, can be found in the Office UI Fabric component distribution: Jquery.SearchBox.js. This also means that you must include a reference to jQuery as the JQuery.SearchBox.js depends on the presence of jQuery.

See the Pen Office UI Fabric - Searchbox by Andrew Connell (@andrewconnell) on CodePen.
