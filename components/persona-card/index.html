Components Persona

The persona card control takes elements from the persona control and adds some action controls so users can interact with the individual such as view their contact options or initiate contact options.

Back to top

Usage

This is a fairly complex control that you should refer to the sample implementation in the demo below for reference on the code to use.

Like the persona control, the persona card control loads a default placeholder image while the referenced person's image is loading or if it fails to load so a broken image is not shown.

You will want to add JavaScript to clear out the placeholder when the control receives focus as shown in the following snippets. An example of this code, using jQuery, can be found in the Office UI Fabric component distribution: Jquery.PersonaCard.js. This also means that you must include a reference to jQuery as the JQuery.PersonaCard.js depends on the presence of jQuery.

See the Pen Office UI Fabric - Persona Card by Andrew Connell (@andrewconnell) on CodePen.

Action Buttons

The actions at the bottom of the card are implemented using the <ul> element and a class .ms-PersonaCard-actions. Each action is it's own <li> with the class .ms-PersonaCard-action. Each action also has a unique id element that is used to enable the cooresponding action's detail pane below it.

The action detail panes are then placed in an <ul> element with the class .ms-PersonaCard-detailChat and wrapped in a <div> with the class .ms-PersonaCard-actionDetailBox. Then each action detail pane is a seperate <li> with the class .ms-PersonaCard-actionDetails and an id that matches the same id in the action buttons. These matching id values are what the script uses to activate each panel. You can indicate which action button is active by adding the class .is-active to the button's <li>.

Presence

The persona card can also show the person's presence state with a little colored circle. With the v1.0.0 release of the Office UI Fabric, it appears the different states are missing from the CSS files, but the sample below shows how you can add the colors in. You can then control the presence by decorating the outter-most <div> with the corresponding CSS class such as .ms-PersonaCard--away.
