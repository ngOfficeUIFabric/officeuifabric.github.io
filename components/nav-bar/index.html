Components Nav Bar

The Office UI Fabric includes a useful navigation control, the nav bar, that is designed to be used at the top of an interface.

Back to top

Usage

Like many other controls, the nav bar control is comprised of many elements wrapped in a <div> decorated with the CSS class .ms-NavBar.

The actual navigation items are implemented within a nexted <ul> list decorated with the class .ms-NavBar-items.

Each navigation item is added as a <li> with the class .ms-NavBar-item. Within it you add a <a> element with the class .ms-NavBar-link to jump to the target page.

The menu is also responsive in the sense that in the case of smaller devices that can't show the full menu, it is collapsed to the standard stacked menu. This is done by adding a <div> with the class .ms-NavBar-openMenu just inside the container and use the provided icon .ms-Icon--menu. You will also need some JavaScript as the example below demonstrates to control the opening and closing of the menu.

You will want to add JavaScript to clear out the placeholder when the control receives focus as shown in the following snippets. An example of this code, using jQuery, can be found in the Office UI Fabric component distribution: Jquery.NavBar.js. This also means that you must include a reference to jQuery as the JQuery.NavBar.js depends on the presence of jQuery.

Submenu

The nav bar control can be combined with the contextual menu control to create a submenu. This is done by including the <ul> element with the .ms-ContextualMenu class within the menu item that contains the submenu. You should also decorate the parent menu item with the class .ms-NavBar-item--hasMenu.

Search Menu

One feature many menus have is the ability to search straight from the navigation. The nav bar supports this as well. Simply add the CSS classes .ms-NavBar-item--search ms-u-hiddenSm to the menu item and then add a text fieldto the inner list item.

Disabled Items

Menu items can also be disabled in the nav bar. This is done by adding the CSS class .is-disabled to the menu item. Refer to the demo above to see an example.

Right-Aligned Items

The nav bar contorl also supports pinning a menu item to the right of all the other items. This is done by adding the CSS class .ms-NavBar-item--right to the menu item. Refer to the demo above to see an example.
