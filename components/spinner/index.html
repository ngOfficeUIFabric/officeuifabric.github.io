Components Spinner

The Office UI Fabric includes spinner working animation that you can use within your applications.

Back to top

Usage

The spinner is mostly implemented with JavaScript and CSS. Unlike other components in the Office UI Fabric, you don't need to add much CSS as it's dynamically created for you along with the elements to implement the spinner. Simply create a <div> element and then initalize it as a spinner. When you initialize it, you can specify if you want it to contain eight (8) or sixteen (16) points in the animation (the default is 8). This is done by passing an instance of the element into a provided function and then telling it to start / stop spinning.

You will want to add JavaScript handle the spinner animation. An example of this code, using jQuery, can be found in the Office UI Fabric component distribution: Spinner.js. This also means that you must include a reference to jQuery as the Spinner.js depends on the presence of jQuery.
