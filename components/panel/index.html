Components Panel

The Office UI Fabric includes a control that you can use to slide in a task-pane like set of content.

Back to top

Usage

To implement the panel, everthing is wrapped within a single <div> decoared with the .ms-Panel class and then optionally include other classes as shown in the examples below to change the size of the panel.

The actual panel is kept within a single nested <div> with the class .ms-Panel-main. Typically you'll want to include a sibling <div> that uses the overlay control to mask out the main part of the UI.

You will want to add JavaScript to toggle the selected check indicator. An example of this code, using jQuery, can be found in the Office UI Fabric component distribution: Jquery.Panel.js. This also means that you must include a reference to jQuery as the JQuery.Panel.js depends on the presence of jQuery.
